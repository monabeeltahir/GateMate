# GateMate

This MATLAB application is a powerful, unified tool specifically engineered for comparative flow cytometry research. It allows researchers to load two distinct datasets, isolate a specific parameter (such as DHR Intensity), apply a singular gating strategy across both samples, and instantly generate alignment-perfect overlays and high-definition statistics bar charts in a standardized format.

By concentrating comparative analysis, gating, and figure generation into a single streamlined dashboard, this application eliminates the tedious manual alignment of figures in third-party software and ensures absolute consistency when comparing distinct population responses before and after exposure.

Figure 1: Complete Comparative Gating Dashboard. The left-most sample (File 1) is gated, and the gating boundaries are instantly applied to the control sample (File 2) and the overlay plots, updating the comparative statistics and bar charts in real-time.
